프로그램 설명
🔔 0부터 9까지의 숫자를 중복 없이 랜덤으로 추출하여 3자리의 숫자를 만든다.
올바른 숫자와 자리를 입력한 경우 : S
올바른 숫자와 올바르지 않은 자리를 입력한 경우 : B
올바르지 않은 숫자를 입력한 경우 : O

🔔 try-catch 문을 사용한 예외 처리
NumberFormatException을 사용하여 정수가 아닌 값을 입력했을 때 "세 자리의 숫자를 입력하세요"라는 문구를 출력한다.

정답 : 123
첫번째 추측 : 145 => 1S 2Out
두번째 추측 : abc => "세 자리 숫자를 입력하세요."(시도 횟수 차감 x)
세번째 추측 : 444 => 3Out
네번째 추측 : 111 => 1S 2B
다섯번째 추측 : 123 => 정답!
